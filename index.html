🎯 充実の機能一覧

サーバー運営に必要な機能を全て搭載

📈

レベリングシステム

チャットするだけで自動的にXPが溜まり、レベルアップ！サーバーの活性度が可視化され、モチベーション向上に貢献します。

/rank /leaderboard 自動付与
🛡️

5重の荒らし対策

アンチレイド、ロックダウン、スローモード、一括削除、BAN/Kick/Timeout...プロ級のセキュリティでサーバーを守ります。

/antiraid /lockdown 自動検知
🎫

チケットシステム

ボタン一つでプライベートチャンネルを作成。サポート対応を効率化し、ユーザーの問い合わせを適切に管理できます。

/ticket setup /ticket close 自動削除
🎭

ロール管理

個別付与から一括操作まで、直感的なコマンドでロールを管理。作成・削除・全員付与もワンコマンドで完了します。

/role add /role all 一括操作
📊

投票機能

最大10選択肢対応の高度な投票システム。リアルタイムで結果が反映され、民主的な意思決定をサポートします。

/poll リアルタイム 自動締切
👋

Welcome/Leave

カスタマイズ可能な入退室メッセージ。変数対応で柔軟な設定が可能で、サーバーの雰囲気作りに最適です。

/welcome /leave DM通知
📉

過疎化チェック

サーバーの活動状況をリアルタイムで分析。過疎度スコア、メッセージ頻度、オンライン率をパネル形式で可視化します。

/kaso check /kaso detail /kaso refresh 分析

🛡️ プロ級荒らし対策

5つの検出システムでサーバーを24時間365日守ります

💬

メッセージスパム検出

7秒間に4件以上のメッセージを検出

1分→5分→1時間の累進タイムアウト
📊

投票スパム検出

1分以内に3回の投票作成を検出

3時間のタイムアウト
🔗

招待リンクブロック

許可されていないチャンネルでの招待リンク検出

自動削除 + 警告
👥

メンションスパム検出

10秒間に3回以上のメンションを検出

10分のタイムアウト
📢

@everyone 検出

管理者以外が@everyoneを使用

30分のタイムアウト
🎵

VCスパム検出

10秒間に3回のVC入室を検出

10分のタイムアウト
🚀

アンチレイド

短時間の大量参加を検出・自動保護

自動ロックダウン + 通知
🧹

一括削除

特定ユーザーのメッセージを一括削除

最大100件を即座に削除
